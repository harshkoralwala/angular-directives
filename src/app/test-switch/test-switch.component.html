Select Gender:

You are a Boy You are a Girl Please select a gender

Explanation:

  • Select gender from dropdown
  • Based on selected gender's internal value,The message will be displayed
    • In code, 2 switch-case clause and 1 default clause used
    • Instead of adding multiple IF statements,its recommanded to use switch statement due to its internal MAP table created while code-compilation process
  • If no gender is selected then default message will be displayed
